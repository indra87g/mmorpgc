A MMORPG game that written in C.

## How To Compile
```sh
# Generate Object
make generateObj

# Compile
make

# Run 
bin/mmorpgc

# Preparation before commit (Delete all binary file)
make clean
```
*To generate object, install `graphviz` first!
```sh
# Install
apt update --fix-missing
apt install graphviz

# It will allow you to use "dot" command
```
